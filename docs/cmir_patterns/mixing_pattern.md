# Mixing Mixed-Integer Inequalities

## Overview

Mixing inequalities is a procedure to generate new valid inequalities for a mixed-integer set \(S \subseteq \mathbb{R}^{m_1} \times \mathbb{Z}^{m_2}\) by combining multiple base inequalities (typically MIR or other known valid inequalities). Unlike simple linear combination, this approach uses the **coefficient polyhedron**, extreme points, and lifting/sequential rules to produce strong, often facet-defining inequalities.

The procedure can be applied in general MIP contexts, including lot-sizing, capacitated facility location, network design, and multiple knapsack problems  .

---

## Base Inequalities

A base inequality has the general form:

\[
f_i(x) + B g_i(x) \ge \pi_i, \quad i \in I
\]

where:

- \(f_i(x)\) is a linear combination of continuous or integer variables,
- \(g_i(x)\) is a linear combination of integer variables,
- \(B \ge 0\), \(\pi_i \in \mathbb{R}\),
- \(x \in S\).

From each base inequality, one can derive a standard **MIR inequality**:

\[
f_i(x) \ge \gamma_i (\tau_i - g_i(x))
\]

with properly defined \(\gamma_i, \tau_i\)  .

---

## Mixing Procedure

Given a subset \(J \subseteq I\) of MIR inequalities:

1. **Identify the subset of base/MIR inequalities** to mix.
2. **Compute extreme points** of the associated coefficient polyhedron \(PC\) (or \(P_\delta(U)\) if grouping independent inequalities).
3. **Form new inequality** using the extreme point coefficients:

\[
\bar{f}(x) \ge \sum_{i \in J} \delta_i (\tau_i - g_i(x)) + \alpha
\]

where \((\delta, \alpha)\) is an extreme point of the polyhedron PC  .

4. **Sequential lifting**: If inequalities are nested or ordered, apply lifting along the sequence of \((\tau_i - g_i(x))\) to strengthen the inequality.
5. **Independent groups**: For base inequalities that are independent w.r.t. a function \(\bar{f}\), apply the mixing procedure within each group, then combine results.
6. **Scaling/Relaxation**: Base inequalities may be scaled or relaxed to tighten the final mixed inequality. This improves the coverage of the convex hull without violating validity  .

---

## Properties

- Mixed inequalities derived from extreme points of \(PC\) are either equivalent to or dominate the inequalities generated by individual MIRs.
- This procedure can produce facet-defining inequalities for the convex hull of certain MIP relaxations.
- The method ensures that the generated inequalities respect the underlying integer/binary structure.

---

## Step-by-Step Application

1. Compute standard MIR inequalities from each base inequality.
2. Identify subsets to be mixed.
3. For each subset, compute extreme points of the coefficient polyhedron.
4. Generate candidate mixed inequalities.
5. If necessary, scale or relax base inequalities to strengthen the mixed inequality.
6. Apply sequential lifting and independent-group combination if applicable.
7. Verify validity against feasible points.
8. Record derivation steps and map to symbolic inequality families.
