## Research report standard

The main report must be organized by symbolic inequality families, not by individual computational instances.

Instance-level data should not dominate the report. It may appear only in:

- a compact evidence table;
- a coverage table;
- an appendix.

For each reported family, include:

1. symbolic statement in the original notation;
2. parameter conditions;
3. derivation source;
4. c-MIR or aggregation certificate;
5. examples of concrete facets covered;
6. validity status;
7. facetness status;
8. completeness status.
